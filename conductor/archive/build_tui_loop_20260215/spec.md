# Specification - Build the core four-panel TUI layout and basic agentic loop shell.

## Overview
Implement the 'loopy' TUI with four panels (Memory, Goals, Self, Activity) using a responsive flex layout. Set up a model-agnostic agentic loop that can execute shell commands.

## Requirements
- Responsive flex layout for tall/wide terminals.
- Four distinct panels updating in real-time.
- Integration with Vercel AI SDK for model-agnostic LLM calls.
- Shell tool for reading/writing files and executing linux commands.
