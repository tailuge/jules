# Technology Stack

## Core Technologies
- **Language:** Bash (Shell Scripting)
- **LLM Interface:** [llm](https://llm.datasette.io/) by Simon Willison (for multi-model interoperability)
- **Environment:** GNU/Linux (standard utilities like `grep`, `sed`, `head`, `tail`, `curl`)

## Infrastructure & Tools
- **Testing:** Integration tests using Bash scripts and standard utilities.
- **Dependency Management:** Runtime verification of tools (`llm`, `pipx`, etc.).
